# <span className="soft">atom.io</span>/foundations/future

`Future` is a `Promise` whose pending value can be replaced.

Use it when callers need a stable promise reference, but the async work that should
resolve that promise may change before it completes.

Construct a `Future` from a promise or from a promise executor.

<Constructor client:load />

## use

Call `use` with a promise to make that promise the active fate. If an older promise
settles later, its result is ignored.

Call `use` with a direct value to resolve the `Future` immediately.

## done

`done` starts as `false` and becomes `true` when the active promise settles.
